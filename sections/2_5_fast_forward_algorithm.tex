\section{The fast-forward modified algorithm} \label{sec:fast_forward_algorithm}

By leveraging the geometric characteristics defined in Theorem~\ref{thm:nstall}, the unpredictable computational trap of the stalling period is resolved. Because the exact duration of the stall is known and the accumulation of the auxiliary variables is strictly linear, the entire stalling phase can be bypassed in a single computational step. By applying constant corrective increments directly to the auxiliary memory variables, we can instantly fast-forward the algorithm's state to the exact iteration where the active constraint breaks at the moment when stalling is detected. Theorem~\ref{thm:nstall} and the baseline Dykstra update are successfully combined in Algorithm~\ref{alg:fastforward}.

\algnewcommand\Inputs{\item[\textbf{Inputs:}]}
\algnewcommand\Outputs{\item[\textbf{Output:}]}

\begin{algorithm}
\caption{Dykstra's projection algorithm for polyhedral sets with fast-forwarding.}
\label{alg:fastforward}
\begin{algorithmic}[1]
    \Inputs $w^\circ, \{a_i, b_i\}_{i=0}^{n-1}, N_{\text{max}}, \epsilon_{\text{stall}}$
    \Outputs Euclidean projection $w^{(m)}$
        \State $w^{(0)} \leftarrow w^\circ$; $e^{(j)} \leftarrow 0\,\forall j\in\{-n,\dots,-1\}$;  $m \leftarrow 0$ 
        \While{$m < N_{\text{max}}$}
            \State $w^{(m+1)} \leftarrow \mathcal{P}_{\mathcal{H}_{[m]}}(w^{(m)} + e^{(m-n)})$\algcomment{Standard polyhedral projection~\eqref{eq:dykstra:proj:poly}}
            \State $e^{(m)} \leftarrow e^{(m-n)} + w^{(m)} - w^{(m+1)}$\algcomment{Standard auxiliary update~\eqref{eq:dykstra:error:poly}}
            \If{$m \ge n-1 \;\textbf{and}\;\twonorm{w^{(m+1-i)} - w^{(m+1-i-n)}} < \epsilon_{\text{stall}}\;\forall i\in\{0,\dots,n-1\}$}\label{alg:start} \algcomment{Stall detected per Definition~\ref{def:stalling}}
                \State Obtain $N_\text{stall}$ and $i_\text{stall}$ from Theorem~\ref{thm:nstall}
                \For{$j \in \{0,\dots,n-1\}$}
                    \State $N_\text{skip}\leftarrow N_\text{stall}$ \textbf{if} $[m-j]\leq i_\text{stall}$ \textbf{else} $N_\text{stall}-1$
                    \State $e^{(m-j)} \leftarrow e^{(m-j)} + N_\text{skip}((w^{(m-j)})^T a_{[m-j]} - b_{[m-j]})a_{[m-j]}$\algcomment{Fast-forward memory variables}
                \EndFor
                \State $m \leftarrow m - i_\text{stall}$\label{alg:end}\algcomment{Adjust iteration counter}
            \Else
                \State $m \leftarrow m + 1$
            \EndIf
        \EndWhile
        \State \Return $w^{(m-1)}$
\end{algorithmic}
\end{algorithm}

The proposed algorithm proceeds identically to the standard Dykstra update until it actively detects a stall. This detection is achieved by verifying the strict stationarity condition $w^{(m)} \approx w^{(m-n)}$ across a full cycle of half-spaces, governed by a small, positive numerical tolerance $\epsilon_{\text{stall}}$. 

Once a stall is detected at iteration $m$, the algorithm interrupts the standard sequential loop and executes its fast-forwarding phase. It first computes the exact number of required stalling cycles, $N_\text{stall}$, and identifies the specific half-space at which the stall terminates, $i_\text{stall}$, by invoking Theorem~\ref{thm:nstall}. For each of the $n$ auxiliary variables in the current cycle, the algorithm computes the constant geometric increment $\Delta e^{(j)} = ((w^{(\text{stall})})^T a_j - b_j)a_j$ and applies a single, aggregated update: $e^{(j)} \leftarrow e^{(j)} + N_{\text{skip}} \Delta e^{(j)}$. The multiplier $N_{\text{skip}}$ is set to $N_{\text{stall}}$ if $i_{\text{stall}}\geq j$ and $N_{\text{stall}}-1$ otherwise, ensuring the internal sequencing aligns with the break condition. 

This single operation artificially ``jumps'' the memory variables forward to their exact mathematical state following the stalling period, entirely bypassing sevral wasted cycles. Finally, the main iteration counter $m$ is adjusted backwards to reflect the index within the cycle where the stall was broken. For practical deployment within an overarching learning framework, Algorithm~\ref{alg:fastforward} is strictly bound by a maximum iteration budget $N_{\text{max}}$. Certain technical implementation details, such as the dynamic updating of the active half-space set $\mathcal{A}$ throughout the algorithm, are omitted here for clarity but are strictly enforced in the software architecture.

Because the proposed fast-forward modification instantly advances the internal memory variables to the exact state they would have eventually reached natively under Dykstra's method, the mathematical integrity of the algorithm is fully preserved (lines \ref{alg:start} to \ref{alg:end}). Consequently, the strong asymptotic convergence properties established by the Boyle-Dykstra theorem~\cite{DYKSTRA} are retained, ensuring that the primary iterates converge to the true Euclidean projection. This is formalised in Corollary~\ref{cor:convergence}.

\begin{corollary}[Convergence of Algorithm~\ref{alg:fastforward}~\cite{DYKSTRA}]
\label{cor:convergence}
The sequence of primary iterates $\{w^{(m)}\}$ generated by Algorithm~\ref{alg:fastforward} converges asymptotically to the true Euclidean optimal solution $w^\star = \mathcal{P}_{\mathcal{H}}(w^\circ)$, such that $\lim_{m\rightarrow\infty}\twonorm{w^{(m)}-w^\star}=0$.
\end{corollary}
