\section{Background on Euclidean projections} \label{sec:euclidean_projections}

The problem of projecting an arbitrary point in a high-dimensional space onto a restricted feasible region is ubiquitous in applied mathematics~\cite{BOYDCONVEX}. This fundamental operation underpins diverse disciplines, from computing restricted least squares in machine learning to the enforcement of strict safety bounds in model predictive control. Formally, we consider a Euclidean space equipped with the standard $\ell_2$-norm, $\twonorm{w}\eqdef (w_1^2+\dots+w_d^2)^{1/2}$, where $d$ represents the dimensions of the decision variable $w$. We define a finite family of $n$ closed convex subsets $\{\mathcal{H}_i\}_{i=0}^{n-1}$ possessing a nonempty intersection $\mathcal{H} := \bigcap_{i=0}^{n-1} \mathcal{H}_i$. Given a proposed unconstrained point $w^\circ \in \mathbb{R}^d$, its exact Euclidean projection $w^\star \reqdef \mathcal{P}_{\mathcal{H}}(w^\circ)$ onto the feasible set $\mathcal{H}$ is obtained by solving the constrained quadratic program (CQP):
\begin{equation} \label{eq:projection}
\begin{array}{ll}
    {\displaystyle \minimise_{w \in \mathbb{R}^d}}  & \frac{1}{2}\twonorm{w - w^\circ}^2 \\
    \text{subject to} & w \in \bigcap_{i=0}^{n-1}\mathcal{H}_i.
\end{array}
\end{equation}

In this thesis, we restrict our analysis to the case where $\mathcal{H}$ is a polyhedral set. This polyhedron can be represented compactly in matrix form as $\mathcal{H}=\{w \in \mathbb{R}^d \mid A w \leq b\}$, with constraint matrix $A \in \mathbb{R}^{n\times d}$ and offset vector $b \in \mathbb{R}^n$. Correspondingly, each individual set $\mathcal{H}_i$ represents a single half-space constraint, written as:
\begin{align}\label{eq:halfspaces_i}
\mathcal{H}_i\coloneqq\{w \in \mathbb{R}^d \mid a_i^T w \leq b_i\}, \quad i=0,\dots,n-1,
\end{align}
where vectors $a_i$ are unit normals to bounding hyperplanes $\partial\mathcal{H}_i\coloneqq\{w \in \mathbb{R}^d \mid a_i^T w = b_i\}$.

While the projection onto a single half-space $\mathcal{H}_i$ is trivial to compute analytically, the joint projection operator $\mathcal{P}_{\mathcal{H}}$ onto the arbitrary intersection of several half-spaces is mathematically intractable, meaning no direct closed-form solution to~\eqref{eq:projection} exists. In traditional settings, the solution to~\eqref{eq:projection} is typically delegated to standard interior-point optimisation software. However, for large $n$, these solvers require introducing additional variables $z\eqdef Aw$ (e.g., \cite{osqp} or~\cite{ocpb:16}) that are projected onto the set $\set{z\in\Rset^n}{z \leq b}$, for which an explicit solution exists~\cite{BOYDCONVEX}. However, for large $n$, this variable augmentation can reduce the computational efficiency, particularly in high-dimensional or time-sensitive applications when the projection is part of an overarching algorithm~\cite{OPTIMNESTEROV,KEMPF20206542}.
